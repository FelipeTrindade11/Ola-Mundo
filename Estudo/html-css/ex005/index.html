Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> e </i> (Não semântico)

Nesta outra, temos um termo em ênfase usando a tag <em> e </em> (Semântico)

Text marcado

Nesta frase, temos um termo marcado usando a tag <mark> e </mark>

Texto grande e pequeno

Nesta frase, temos uma parte do texto grande usando <big> e </big> que está obsoleto e uma parte do texto pequeno <small> e </small> usando que não está obsoleto

Texto deletado

Podemos marcar um texto como excluido para lermos mas não considerarmos, utilizando as tags <del> e </del>

Texto inserido

Podemos também marcar um texto como inserido para darmos ênfase no texto que foi adicionado depois, utilizando as tags <ins> e </ins>
